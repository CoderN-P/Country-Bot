# Configurechannel / Unconfigurechannel

## Configurechannel

Set up a channel to receive updates about the bot

### Usage

```text
.configurechannel <channel>
```

## Unconfigurechannel

Make a channel not receive updates about the bot

```text
.unconfigurechannel <channel>
```
